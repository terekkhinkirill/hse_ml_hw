# Домашнее задание 1
Домашнее задание состоит из 6 частей:
1. **EDA:** Подсчет базовых статистик датасета, очистка данных, визуализация попарных распределений и матрицы корреляций.
2. **Обучение модели на вещественных признаках** Обучение линейной регрессии, Lasso-регрессии, ElasticNet. Подсчет метрик качества MSE и $R^{2}$. Обучение модели с использованием GridSearchCV.
3. **Обработка категориальных признаков** Использование OneHotEncoder. Обучение Ridge-регрессии.
4. **Бизнесовая** Подсчет бизнес-метрик и сравнение моделей.
5. **Создание интерактивного приложения на Streamlit** Сохранение пайплайна модели. Реализация streamlit-приложения.
6. **Оформление репозитория**

### Какие результаты были получены:
- **Базовая линейная регрессия (только числовые признаки):** $R^{2}$ = 0.6 — неплохая отправная точка
- **Ridge-регрессия с категориальными признаками:** Наилучший результат — $R^{2}$ = 0.78 значительно улучшился относительно базовой модели
- **Бизнес-метрики:**
  - Модель Ridge показала лучшие результаты по доле прогнозов с ошибкой ≤10%
  - Та же модель лучше всего справилась с асимметричной метрикой, где недопрогноз хуже перепрогноза
